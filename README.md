#수정 방법

1. 깃헙에 로그인

2. 좌측에서 "cha-145692/artmug_v1" 클릭

3. 소스 수정은 index.html > 우측에 연필버튼 > 수정 필요한 부분 수정 > Commit changes 버튼 클릭 > Commit changes 버튼 클릭

4. 이미지 추가는 img 폴더 > Add file > Upload files > 파일 추가 > Commit changes 버튼 클릭 > Commit changes 버튼 클릭
   4-1. 이미지 파일 명은 영어,숫자, _, - 로만. (오류방지)

6. 안 쓰는 부분은 <!-- 안 쓸 부분 --> 으로 감싸서 노출 숨기기

7. push 후 최대 2분 이내로 자동으로 배포
